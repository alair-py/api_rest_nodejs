# api_rest_nodejs
API REST em Node.js utilizando Sequelize com MySQL,
consumo da API com Axios e tokens de acesso com JWT.
Várias outras funcionalidades, autenticação de usuários, encriptação de senhas com hash (bcryptjs)
E outros mais.
